    Data Manipulation Techniques

  • Filtering: Extracting specific data based on conditions or criteria (e.g., retrieving all products priced above $50).
  • Sorting: Arranging data in a particular order, such as ascending or descending (e.g., sorting by name, date, or price).
  • Grouping: Combining data based on shared attributes (e.g., categorizing orders by customer or region).
  • Aggregation: Summarizing data using operations like sum, average, or count (e.g., calculating total sales for a month).
  • Transforming: Modifying data format or structure for easier analysis (e.g., converting a date to a different time zone or changing a currency format).
  • Joining: Merging two or more datasets based on a shared key (e.g., combining customer data with order history).
  • Cleaning: Detecting and rectifying errors, inconsistencies, or missing values in data (e.g., replacing null values or correcting typos).
